Program di atas berfungsi untuk menghitung rata-rata nilai siswa. 
Cara kerjanya yaitu pengguna diminta memasukkan jumlah siswa terlebih dahulu, lalu program menggunakan perulangan for untuk meminta input nilai setiap siswa satu per satu.
Setiap nilai yang dimasukkan akan dijumlahkan ke variabel temp. 
Setelah semua nilai diinput, program menghitung rata-ratanya dengan membagi total nilai (temp) dengan jumlah siswa (jumlahSiswa).
Terakhir, program menampilkan hasil rata-rata nilai seluruh siswa di layar.
